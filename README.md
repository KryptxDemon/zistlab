# ZIST Static Pages

This project is now a multi-page static site built with plain HTML and CSS.

## Pages

- `/index.html`
- `/login.html`
- `/signup.html`
- `/forgot-password.html`
- `/dashboard.html`
- `/library.html`
- `/add-media.html`
- `/media-detail.html`
- `/quiz-hub.html`
- `/quiz-session.html`
- `/feed.html`
- `/vocabulary.html`
- `/insights.html`
- `/profile.html`
- `/not-found.html`

Each page has its own CSS file in the project root.

## Stack

- HTML5
- CSS3
- Vite (static dev server/build)

## Run

```sh
npm install
npm run dev
```

Open `http://localhost:8080`.
